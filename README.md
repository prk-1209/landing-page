A second basic webpage.

This is another exercise from The Odin's Project where skills on Terminal use, HTML combined with CSS, Git and GitHub are practiced.
Therefore I have been using OS computer, VisualCode and a lot of code and images from previous TOP tasks (foundations/flex) adjusted to fit the desired outcome.

First I created the repository on GitHub and cloned it locally. I worked the whole time locally and when everything was finished I git added, committed and pushed. I recognized that by doing so I lost all the visibility, so in order to do it right I started the project for a second time and deleted all the code from the files (instead of removing the whole repository).

The beginning step was creating containers in the HTML file (with this project I realized the importance of containers - everything can behave differently just by surrounding one item). Than I used a lot of code and some images from previous excesses - css excesses/foundations/flex from The Odin Project. For the hero section, used and modified code from Flex exercises 07 and 05; for the mid section, used Flex 04; and for the action section, used Flex 05. The image in 'I'm a card' comes from css-exercises/foundations/block-and-inline/02-margin-and-padding-2.

Further I worked on the CSS file starting with the given information about fonts, colors and backgrounds. Than I looked at how to structure everything in the best, shortest and simplest way possible (this took some time - honestly, the second time writing the code went much more efficient and it looks better now!).I also forgot to commit once before the last commit, so the changes are more than usual and decided to leave it like that. 

Soo yes, there are some new things that I used which I have found on the internet, like this one:

.container
    max-width: 900px;
    margin: 0 auto;

It helped a lot for centralizing and setting a width to the whole page.
Also, the part '.item > div' I found on the internet and it is a handy one because it targets deeper elements if you have more containers. 

That's all I can say! In general, a very helpful exercise - I could practice and learn a lot from it. Looking forward to seeing what's next!
